# 리덕스

    - 여러 페이지, 여러 컴포넌트 사이에 한개의 변수가 동일하게 관리되겠금 처리하는 기술
    - 전역 변수 관리 기술
    - 특징
        - 모든 시퀀스는 단방향으로만 진행
        - 컴포넌트 -> 액션(+버튼 클릭 -> 숫자 증가 액션) ->
          -> reducer전달(상태 저장 및 액션에 대응하는 처리, 값을 대체한다(업데이트x, 바꿔치기))
          -> store 전달(저장소) -> 컴포넌트 전달
    - 쉽게 개발
        - MobX, Recoil 대체 처리 가능
        - 훨씬 수월하게 개발 가능
